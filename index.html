my resume

Skill

개인적으로 충분한 경험을 가지고 웹 서비스에서 활용될 기술과 다룰 수 있는 아트 툴을 기재하였습니다.

HTML 5

웹 표준과 웹 접근성을 준수하는 HTML5 정보 구조 설계로 웹사이트 퍼블리싱을 하고 있습니다. 퍼블리싱 작업 전에 ‘기능정의서’를 만들어 폰트, 컬러, 사이즈 정보, 상호작용 여부, 와이어프레임까지 꼼꼼하게 정리해두고 제계적인 퍼블리싱 작업을 수행합니다.

CSS 3

CSS 코딩은 IE를 지원을 충분히 고려하는 크로스브라우징(Cross Browsing)을 위해 안정적인 css 속성을 사용해서 HTML Elements를 배치하고 심미적으로 디자인 합니다. CSS3에서 새롭게 지원하는 속성을 활용해서 다양한 CSS 배치와 디자인을 다룹니다.

JS

네비게이션, 탭 메뉴, 어코디언 등 웹 퍼블리싱에 반드시 필요한 상호작용을 위한 제이쿼리는 플러그인을 사용하지 않고 순수만 제이쿼리 코딩으로 제작합니다. 필수적인 자바스크립트 활용이 가능하며 주로 제이쿼리와 병행해서 활용하고 있습니다. (if 조건문, for 반복문 등)

TEXT EDITOR

웹 퍼블리셔로서 텍스트 에디터는 Visual Studio Code를 메인으로 사용하고 있으며 상황에 따라 Adobe Brackets를 병행해서 사용하기도 합니다. 부분 수정 또는 HTML5 와이어 프레임 구조를 설계할 때 수월한 퍼블리싱을 위해 NotePad++를 적극 활용합니다.

RESPONSIVE WEB

반응형 웹 제작의 크로스브라우징을 위해 안정적인 기존 방식인 float, position, Media Query를 활용한 경험을 가지고 있습니다. 필요에 따라 플렉스(Flex)와 그리드(Grid)로 반응형 웹 사이트 제작이 가능합니다. 모바일 웹 앱 제작의 경우 Flex와 Grid를 활용해서 제작할 수 있습니다.

MOBILE WEB APP

모바일 우선을 위해 모바일 웹 앱 제작 경험과 UI/UX 측면에서 심미적인 요소와 모바일 사용자 경험의 최적화를 위해 효율적인 콘텐츠를 배치를 우선적으로 생각합니다. 또한 모바일 대응을 위한 플랫폼으로 반응형 웹, 모바일 웹, 모바일 웹 앱, 네이티브 앱에 대한 이해도가 있습니다.

GRAPHIC

배너, 이벤트, 로고 등의 웹 그래픽 아트웍은 포토샵과 XD, 일러스트 위주로 사용합니다. 포토샵과 XD는 웹사이트 UI 디자인 작업 시 충분히 활용 가능하며, 영상 편집 및 제작과 트랜지션 경험은 프리미어를 사용하고 그래픽 아트웍은 타블렛 장비를 곁들여 사용합니다.

XD

웹사이트 UI 디자인 시 XD와 포토샵을 활용하고 있습니다. 다양한 효과가 돋보이는 요소 및 사용자 경험을 고려한 만족도 높은 디자인으로 설계할 수 있습니다. 또한 디자인 측면과 기능적인 측면에서 상호작용 시 불편함을 최소화하며 심미적으로도 훌륭한 디자인 요소를 제공합니다.

Publishing / Web & Design Portfolio

carpe diem!
